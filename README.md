Sistema Runner

O Sistema Runner é uma aplicação de linha de comando desenvolvida para facilitar a execução de aplicações Java, abstraindo configurações do ambiente.
É composto pelos módulos assinatura (CLI), assinador.jar (processamento) e simulador (integração).
Permite realizar operações de assinatura e validação de forma simplificada, com validação de parâmetros.
As funcionalidades são simuladas, não envolvendo criptografia real.
